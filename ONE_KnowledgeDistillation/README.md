# Knowledge Distillation with ONE (On-the-Fly Native Ensemble)

This folder contains my experiments based on the paper:
- Xu et al., *Knowledge Distillation On the Fly Native Ensemble (ONE)*, NeurIPS 2018.  
- Original implementation: [Lan1991Xu/ONE_NeurIPS2018](https://github.com/Lan1991Xu/ONE_NeurIPS2018)

---

## Contents
- `experiments.ipynb` → my notebook where I modify initial parameters and change the NN structure.
- `configs/` → optional parameter configuration files.
