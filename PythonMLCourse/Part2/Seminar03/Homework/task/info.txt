Вход: Файл входного тензора tensor.npy размерности batch x h x w x c_in, 
      файл ядра свертки kernel.npy размерности kh x kw x c_in x c_out, 
      файл свободных членов bias.npy размерности c_out 
(эти три файла - записаны с помощью numpy.save(), 
соответственно прочитать их можно с помощью numpy.load()), 
а также задача task.csv, в которой задан параметр stride (в формате csv с заголовком).

Выход: Файл с названием seminar03_conv.npy, в котором должен быть записан результат применения 
свертки с вектором свободных членов к входному тензору с заданным параметром stride в режиме 
'VALID', в формате NHWC. Полученный результат p сохраняется с помощью стандартной функции numpy: 
numpy.save('seminar03_conv.npy', p, allow_pickle=False)
